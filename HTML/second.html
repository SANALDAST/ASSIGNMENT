whatsapp web

WhatsApp was officially made available for PCs through a web client, under the name WhatsApp Web, in late January 2015 through an announcement made by Koum on his Facebook page: "Our web client is simply an extension of your phone: the web browser mirrors conversations and messages from your mobile deviceâ€”this means all of your messages still live on your phone". The WhatsApp user's handset must still be connected to the Internet for the browser application to function. All major desktop browsers are supported except for Microsoft Internet Explorer. WhatsApp Web's user interface is based on the default Android one. As of January 21, 2015, the desktop version was only available to Android, BlackBerry and Windows Phone users. Later on, it also added support for iOS, Nokia Series 40, and Nokia S60 (Symbian).[56][57] An unofficial derivative called WhatsAppTime has been developed, which is a standard Win32 application for PCs and supports notifications through the Windows notification area.[58] There are similar solutions for Mac OS X, such as the open-source ChitChat[59][60][61] and multiple wrappers available in the App Store.[citation needed]
